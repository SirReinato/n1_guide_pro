Instalação Python

Esse manual tem como passso a passo a instalação do python

1 - Baixar o Instalador do Python

Visite o site oficial do Python (https://www.python.org) e baixe a versão mais recente do Python para Windows. Certifique-se de escolher o instalador correto para a arquitetura do seu sistema (32 ou 64 bits).

2 - Executar o Instalador

Após baixar o instalador, execute o arquivo. Na primeira tela, marque a opção 'Add Python to PATH' e clique em 'Install Now' para iniciar a instalação.

3 - Aguardar a Instalação

Aguarde até que o Python seja instalado no seu sistema. Isso pode levar alguns minutos.

4 - Verificar a Instalação

Após a instalação ser concluída, abra o 'Prompt de Comando' (cmd) e digite 'python --version' para verificar se o Python foi instalado corretamente.

5 - Instalar o PIP (Opcional)

O PIP, gerenciador de pacotes do Python, já vem incluído na instalação padrão. Para verificar, digite 'pip --version' no terminal.

6 - Testar o Python

Abra o 'Prompt de Comando', digite 'python' e pressione Enter. O interpretador do Python deve ser iniciado, permitindo que você digite comandos diretamente.
