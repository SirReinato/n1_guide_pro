Instalação Oracle Cliente 18.5

Este manual detalha os passos necessários para a instalação do cliente Oracle versão 18.5 e configuração para o SISFAF.

1 - Instalar o vcredist_x86

Acesse o site da Microsoft (https://www.microsoft.com/en-us/download/details.aspx?displaylang=en&id=14431), baixe o arquivo 'vcredist_x86.exe' e instale-o seguindo as instruções.

2 - Baixar os Pacotes Oracle

Faça o download dos pacotes 'Basic Package' e 'ODBC Package' na versão 18.5.0.0.0 do site da Oracle ou no diretório '\\mds.net\mds\SE\DTI\CGSS\SUPORTE\APLICACOES\Oracle\Oracle_18.5'.

3 - Criar Diretórios no C:

Crie uma pasta 'Oracle' no 'C:/'. Extraia os conteúdos dos pacotes baixados para 'C:/Oracle' e crie a subpasta 'bin' em 'C:/Oracle/instantclient_18_5'. Copie o arquivo 'oci.dll' para 'C:/Oracle/instantclient_18_5/bin'.

4 - Instalar o Driver ODBC

Execute como administrador o arquivo 'odbc_install.exe' em 'C:/Oracle/instantclient_18_5' para instalar o driver ODBC.

5 - Configurar Variáveis de Ambiente

No editor de variáveis do sistema, adicione o caminho de instalação do cliente Oracle ao 'Path' e crie uma variável 'TNS_ADMIN' com o diretório do SISFAF.

6 - Editar o Registro do Windows

No 'Editor do Registro', limpe as variáveis existentes e configure 'ORACLE_HOME' e 'NLS_LANG' com os valores apropriados.

7 - Configurar o Driver ODBC

Abra o 'ODBC Data Sources (32-bit)', selecione 'DSN de Sistema' e configure o 'Oracle in instantclient_18_5' com os dados do SISFAF.

8 - Validar o Acesso ao SISFAF

Solicite ao usuário que acesse o programa SISFAF e valide a conexão com o banco de dados.
